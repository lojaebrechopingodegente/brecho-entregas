🛍️ Registro de Vendas - Live

Gerencie suas vendas em tempo real

🔧 Configuração do Supabase

⚠️ Execute este SQL no Editor do Supabase primeiro:

create table if not exists vendas_live ( id uuid default gen_random_uuid() primary key, linha_id integer not null unique, item integer, valor decimal(10,2) default 0, codigo text, comprador text, fila text, total decimal(10,2) default 0, created_at timestamp with time zone default timezone('utc'::text, now()), updated_at timestamp with time zone default timezone('utc'::text, now()) ); create index if not exists idx_vendas_live_linha on vendas_live(linha_id); create index if not exists idx_vendas_live_comprador on vendas_live(comprador);

* Pressione ENTER para navegar entre os campos. Os dados são salvos automaticamente.
