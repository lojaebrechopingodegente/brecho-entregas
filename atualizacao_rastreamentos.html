🛠️ Escolha um Método de Verificação

💡 Dica: Se um método não funcionar, tente outro. Todos acessam as mesmas APIs.
🖥️ Método 1: PowerShell (Recomendado)
Copie o script abaixo, cole no PowerShell como Administrador e execute. O resultado aparece na tela para você copiar. 
# Cole no PowerShell (Admin) - Execute linha por linha se der erro
$key = "SUA_CHAVE_AQUI"
$url = "https://ytnfjzngalfapfnbobfm.supabase.co"
$h = @{ "apikey" = $key; "Authorization" = "Bearer $key" }

# Testar conexão
Invoke-RestMethod -Uri "$url/rest/v1/superfrete_config?limit=1" -Headers $h

# Buscar tokens
$sf = (Invoke-RestMethod -Uri "$url/rest/v1/superfrete_config?select=api_token&is_active=eq.true&limit=1" -Headers $h)[0].api_token
$me = (Invoke-RestMethod -Uri "$url/rest/v1/melhorenvio_config?select=access_token&is_active=eq.true&limit=1" -Headers $h)[0].access_token

# Buscar entregas pendentes
$sol = Invoke-RestMethod -Uri "$url/rest/v1/solicitacoes_pg?select=id,numero_solicitacao,codigo_rastreio,forma_envio,clientes_pg(nome)&status=eq.enviado&limit=10" -Headers $h

# Verificar cada uma (exemplo com SuperFrete)
foreach ($s in $sol) {
    $r = Invoke-RestMethod -Uri "https://api.superfrete.com/v1/trackings/$($s.codigo_rastreio)" -Headers @{ "Authorization" = "Bearer $sf" } -ErrorAction SilentlyContinue
    if ($r.status -eq "delivered") { Write-Host "ENTREGUE: $($s.codigo_rastreio)" -ForegroundColor Green }
    else { Write-Host "Transito: $($s.codigo_rastreio)" -ForegroundColor Yellow }
    Start-Sleep -Milliseconds 500
}
🌐 Método 2: cURL no Prompt de Comando (CMD)
Funciona no CMD do Windows 10/11 sem instalar nada. Execute comando por comando. 
# Abra o CMD e execute:
curl -X GET "https://ytnfjzngalfapfnbobfm.supabase.co/rest/v1/superfrete_config?select=api_token&is_active=eq.true&limit=1" -H "apikey: SUA_CHAVE" -H "Authorization: Bearer SUA_CHAVE"

# Buscar entregas pendentes:
curl -X GET "https://ytnfjzngalfapfnbobfm.supabase.co/rest/v1/solicitacoes_pg?select=codigo_rastreio,forma_envio,clientes_pg(nome)&status=eq.enviado&limit=10" -H "apikey: SUA_CHAVE" -H "Authorization: Bearer SUA_CHAVE"
