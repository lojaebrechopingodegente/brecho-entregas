🖥️ Rastreamento via PowerShell

Use o PowerShell para rastrear sem limitações de CORS. Baixe o script abaixo e execute no seu computador.

# 1. Baixe o script rastreamento.ps1 # 2. Abra o PowerShell como Administrador # 3. Navegue até a pasta do arquivo: cd C:\caminho\para\o\script # 4. Execute a verificação automática: .\rastreamento.ps1 -Acao verificar # 5. Ou rastreie um código específico: .\rastreamento.ps1 -Acao rastrear -CodigoRastreio "SF123456789BR"
