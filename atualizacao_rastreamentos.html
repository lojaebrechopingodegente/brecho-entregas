  2. Clique em "📋 Copiar Script" (código limpo, sem HTML)
  3. Cole no PowerShell como Administrador e execute
  4. O script irá: buscar tokens → buscar entregas pendentes → consultar APIs → retornar resultado
  5. O resultado JSON será copiado automaticamente!
  6. Vá direto para aba "Resultado" e cole (Ctrl+V)
