🔄 Conciliador de Entregas

Atualiza status das solicitações baseado nos rastreamentos

✅ Novo: Este script atualiza automaticamente o status das solicitações quando encontra rastreamento correspondente!
📋 O que este script faz:
  1. Busca solicitações com status "pendente" ou "enviado"
  2. Compara com rastreamentos existentes
  3. Atualiza status para "conciliado" (ou "entregue" se aplicável)
  4. Atualiza campo "atualizado_em" com data/hora atual
  5. Retorna relatório completo da conciliação
